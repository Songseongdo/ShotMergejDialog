FSX Log Tail Sender install notes

1. EXE user
- Python is not required to run the built EXE.
- Run:
  dist\FsxLogTailSender.exe
- Network access to the server URL is required.
- The app reads:
  FSXLogFile.txt
  LastShot.CSV

2. Python source user
- Python 3.10 or newer is recommended.
- Python 3.11.9 was used for the current build.
- Required Python package:
  requests
- Install command:
  python -m pip install requests
- tkinter is also required, but it is included in the standard python.org Windows installer.

3. Rebuild EXE
- Required build package:
  pyinstaller
- Install command:
  python -m pip install pyinstaller
- Build command:
  python -m PyInstaller --noconsole --onefile --name FsxLogTailSender --distpath dist --workpath build --specpath . --noconfirm --clean fsx_log_tail_sender.py

4. Server side
- This sender posts to:
  http://127.0.0.1:5000/shot
- If you run server_viewer_tabs.py from Python, install:
  python -m pip install flask requests
